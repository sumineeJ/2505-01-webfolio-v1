🧾 봄바람재활의학과 BI / 브랜딩
🍃 겨울을 지나 봄을 맞이하는 따뜻한 바람처럼, 환자 곁에 머무는 브랜드
📍 프로젝트 개요
  • 대상: 가상의 재활의학과 의원
  • 목표: 몸과 마음이 지친 환자에게 봄바람처럼 다가갈 수 있는 따뜻한 병원의 첫인상 설계
🎯 기획 포인트
  • 병원 이름부터 직접 정하고, 로고 방향성까지 처음부터 끝까지 직접 설정
  • ‘회복’과 ‘위로’의 상징인 봄바람을 표현하기 위해 곡선의 나뭇잎을 기본 모티브로 사용
  • 벚꽃과 의료 십자가를 결합해 직관성과 상징성 동시 전달
🎨 디자인 특징
  • 로고 심벌: 곡선형 나뭇잎 + 의료 십자가 + 벚꽃 형상
  • 메인 타이틀: 날렵한 붓글씨 스타일로 부드럽고 정감 있게
  • 서브 타이틀: 본 색상보다 진한 컬러 + 명확한 폰트
  • 컬러 선택: 전체적으로 부드럽고 안정적이나 조금 더 힘 있는 색도 고려했을 것
🤔 작업 중 고민

처음 BI 디자인에 도전한 프로젝트라 막막함부터 시작했어요.
추상적인 스타일을 추구했지만, 오히려 혼란을 주기도 했고요.
유치하게 보일까 걱정했던 순간도 있었고, 아이덴티티를 시각화하는 일의 무게를 크게 느꼈어요.
